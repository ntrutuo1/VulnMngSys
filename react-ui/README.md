# VulnMngSys React UI

Modern React interface for VulnMngSys scan flow.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Notes

- This UI currently simulates scan results in-browser.
- You can replace `simulateScan` in `src/catalog.js` with real API calls to your Python scan engine.
